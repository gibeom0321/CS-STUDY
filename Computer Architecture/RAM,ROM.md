# 메모리
## RAM
-  정의

RAM(read access memory)는 읽고 쓸수 있는 메모리로 전력이 공급될때만 정보가 저장되는 휘발성 메모리이다. 메인메모리에 사용된다.
-  종류

RAM의 종류로는 대표적으로 DRAM과 SRAM이 있다. DRAM은 동적 렘으로 계속 리셋되서 refresh가 필요하다. SRAM은 정적 렘으로 전력이 공급되는 한 값이 사라지지 않는다. 메인메모리에 사용된다.

- DRAM vs SRAM

DRAM은 트렌지스터 1개, 콘덴서 1개로 이루어져있고, SRAM은 트렌지스터 6개로 이루어졌다. 그래서 DRAM이 더 효율적으록 공간을 쓸수 있고, 트랜지스터가 적기 때문에 더 싸다는 장점이 있다.
SRAM은 DRAM에 비해 속도가 빠르고 기억이 휘발될 위험이 없다는 장점이 있다.
## ROM
-  정의
ROM(read only memory)는 읽을수만 있는 메모리인데 전력이 끊어져도 정보다 계속 있는 메모리이다. 부팅 프로그램이나 펌웨어가 저장된다.

- 종류

ROM -> PROM -> EPROM -> EEPROM 순으로 발전됨.

ROM: 수정 X

PROM: 한번만 수정 가능

EPROM: 자외선을 이용하여 여러번 수정 가능

EEPROM: 전기를 이용하여 여러번 수정 가능
